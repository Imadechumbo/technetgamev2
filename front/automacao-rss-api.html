Automação de últimas notícias por RSS/API

Guia prático da estrutura já embutida no projeto para alimentar cards e blocos automáticos sem redesenhar as páginas.

Como funciona

O projeto agora inclui assets/js/feeds.js?v=20260330-v20, que detecta blocos com data-feed-block e renderiza itens localmente. Você pode começar com fallback local e depois ligar um feed real.

Formatos aceitos

  • JSON/API: array simples ou objeto com chave items
  • RSS: com proxy/backend ou adaptador como rss2json
  • Fallback local: mantém a home preenchida enquanto a integração real não entra

Modelo JSON sugerido

  • title
  • url ou link
  • summary ou description
  • source
  • date

Exemplo de uso

data-feed-type="json"

data-feed-url="/api/steam-news.json"

data-feed-label="Notícias da Steam"

data-feed-limit="6"

Se quiser evitar CORS e dependências de terceiros, o caminho mais robusto é gerar JSON no seu backend e servir localmente no mesmo domínio.
